Hey neighbor — welcome to the sim. We made a tiny site so everyone in the building gets a fair shot at playing. Scan, check in, or grab a slot for later today.

If you're heading down now

  1. Scan the code with your phone camera
  2. Enter your name (it'll remember next time)
  3. Tap "I'm playing right now"
  4. Pick how long you'll play and whether you're cool with company

If you want a slot later today

  1. Scan the same code
  2. Tap "Schedule a slot today"
  3. Pick your start time (anytime left today)
  4. Set your duration and join preferences — done

House rules

  • Day-of bookings only — no week-out hoarding, keeps it fair
  • If someone reserved exclusive time, please respect it
  • Cancel your slot if plans change so others can grab it
  • Honor system — be cool, clean up, have fun
